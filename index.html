열요금제 비교 시뮬레이터

월별 입력(12월/익년 1월/익년 2월) + 연도별 요약 + 입력기간 합산 비교 (입력 단위: Gcal)

단일요금 단가
수요관리형(피크 07~10)
수요관리형(그 외)
분기점(p*)
p* = (단일단가 − 비피크단가) / (피크단가 − 비피크단가)
전체 합산 기준으로 (수요관리시간대 사용량 ÷ 총 열사용량) 비율이 p*보다 크면 단일요금이 유리합니다.
※ VAT별도 금액 기준으로 표시됩니다.
※ 단위 참고: 1 Gcal = 1,000 Mcal

월별/연도별 입력

각 연도별 12월~익년 2월(동절기) 입력

입력 안내
• 입력 단위는 Gcal이며, 소수 1자리까지 입력할 수 있습니다.
• 수요관리시간대(07~10시) 사용량은 Qp로 입력합니다.
※ 현재 값은 예시입니다. 실제 사용량을 입력해 주세요.
